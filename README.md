# Sami's Portfolio (3D)

3D portfolio website built with:
- Vite + React
- three.js + @react-three/fiber

## Live Demo
(Add your Vercel link after deploy)

## Features
- 3D background scene with interactive project panels
- Clean hero-first one-page layout
- Projects section updates when a 3D panel is selected
- CV download link

## Run locally
npm install
npm run dev

## Build
npm run build
npm run preview
